## ðŸš€ Features
- Dynamic hydration goal
- Rotary dial input to declare water intake
- Confirmation button to subtract intake from goal
- Lottie animation background for visual engagement
- Save water intake and goal to SharedPreferences
- Reset daily goal at midnight
- input carousel with pre-defined values (5, 15, 25, 33, 50, 75, 100, 125, 150, 200 cl)
- Dynamic daily goal based on time elapsed since midnight
